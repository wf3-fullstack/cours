    => PERSONA CLIENT
    => ENSUITE CREER UN/DEUX PERSONA VISITEUR(S)

    FAIRE LA LISTE DES FONCTIONNALITES SUR CHAQUE TEMPLATE

    CREER LES TEMPLATES HTML, CSS, JS, etc...

    CREER LE REPO GITHUB POUR LE PROJET
    INSTALLER SYMFONY

    INTEGRER LES TEMPLATES DANS SYMFONY
